# PRD: 하루치 - 햄스터 다마고치 게임

## 📋 제품 기획서 (Product Requirements Document)

### 1. 무엇을 만들 것인가?

**제품명**: 하루치 (Haruchi) - 웹 기반 햄스터 다마고치 게임

**제품 설명**: 
90년대 다마고치를 모티브로 한 웹 게임으로, 귀여운 햄스터 "하루치"를 키우며 돌봄을 통해 레벨업하고 성장시키는 게임입니다.

**핵심 가치**:
- **노스탤지어**: 90년대 다마고치의 감성을 현대 웹 기술로 재현
- **간편함**: 링크만 공유하면 바로 플레이 가능 (설치 불필요)
- **반응형**: PC와 모바일 모두에서 최적화된 경험
- **게이미피케이션**: 레벨업, 경험치 시스템으로 지속적인 동기부여

### 2. 문제 정의

**해결하려는 문제**:
- 모바일 게임 설치의 번거로움
- 복잡한 게임 시스템의 진입 장벽
- 단순하지만 중독성 있는 캐주얼 게임의 부재
- 노스탤지어를 느낄 수 있는 웹 게임의 부족

**타겟 사용자**:
- 90년대 다마고치를 경험한 세대
- 간단한 캐주얼 게임을 즐기고 싶은 사용자
- 링크 공유로 쉽게 즐길 수 있는 게임을 원하는 사용자

### 3. 성공 기준 (Success Metrics)

#### 핵심 성공 지표 (KPI)

**사용자 참여도**:
1. **일일 활성 사용자 (DAU)**: 주간 평균 100명 이상
2. **평균 세션 시간**: 5분 이상
3. **재방문률**: 주간 재방문 사용자 30% 이상
4. **레벨 달성**: 평균 레벨 5 이상 달성 사용자 50% 이상

**기술적 성공 지표**:
1. **페이지 로딩 시간**: 3초 이내
2. **모바일 호환성**: 주요 모바일 브라우저에서 정상 작동
3. **데이터 저장 안정성**: localStorage 데이터 손실률 1% 미만
4. **성능 점수**: Lighthouse 성능 점수 80점 이상

**비즈니스 성공 지표** (Pro 버전):
1. **프리미엄 전환율**: 기본 버전 사용자의 5% 이상이 Pro로 전환
2. **Pro 기능 사용률**: 자동 레벨업, 씨앗 농장 등 Pro 기능 활성 사용자 70% 이상

#### 부가 성공 지표
- 소셜 공유 횟수 (카톡, SNS 등)
- 사용자 피드백 및 만족도
- 버그 리포트 및 안정성

### 4. 핵심 기능

#### 4.1 기본 기능 (Basic Tier)

**P0 - 필수 기능**:
1. **햄스터 돌봄 시스템**
   - 밥 주기 (EXP +10)
   - 물 주기 (EXP +5)
   - 쓰다듬기 (EXP +15)
   - 집, 그릇, 물통 클릭 액션

2. **레벨업 시스템**
   - 경험치(EXP) 축적
   - 레벨업 시 최대 EXP 증가
   - 레벨 표시 및 진행률 바

3. **상태 시스템**
   - 기분/상태에 따른 표정 변화
   - 애니메이션 및 리액션
   - 배고픔, 목마름, 행복도 등

4. **오디오 시스템**
   - 오프닝 배경음
   - 게임 BGM
   - 효과음 (선택적)

5. **데이터 저장**
   - localStorage 기반 저장
   - 게임 상태 자동 저장
   - 통계 데이터 저장

6. **통계 화면**
   - 플레이 일수
   - 총 액션 횟수
   - 레벨, EXP 정보
   - 출석 캘린더

#### 4.2 프로 기능 (Pro Tier)

**P1 - Pro 전용 기능**:
1. **자동 레벨업**
   - 노션 연동으로 할 일 체크 시 자동 EXP 지급
   - 폴링 시스템 (30초 간격)

2. **씨앗 농장 (Streak Calendar)**
   - 연속 플레이 일수 시각화
   - 씨앗 → 해바라기 성장 애니메이션
   - 출석 동기부여

3. **똑똑한 잔소리 (Smart Feedback)**
   - 달성률에 따른 하루치 대사 변화
   - 칭찬 ↔ 시무룩 반응

4. **통합 대시보드**
   - 목표, 습관, 일기 한 화면
   - 모바일 최적화 뷰

### 5. 기술 스택

- **프론트엔드**: HTML5, CSS3, Vanilla JavaScript
- **데이터 저장**: 브라우저 localStorage
- **배포**: Vercel (정적 호스팅)
- **폰트**: Press Start 2P (게임보이 스타일), Noto Sans KR
- **성능 모니터링**: Vercel Speed Insights

### 6. 제약사항

- **백엔드 없음**: 서버 없이 클라이언트만으로 동작
- **데이터 동기화 불가**: 기기/브라우저별로 독립적인 저장
- **실시간 멀티플레이어 불가**: 단일 사용자 게임
- **오프라인 지원**: 인터넷 연결 필요 (Vercel 배포)

### 7. 우선순위

**P0 (필수 - v1.0)**:
- 기본 돌봄 시스템
- 레벨업 시스템
- 상태 및 애니메이션
- 오디오 시스템
- 데이터 저장
- 통계 화면

**P1 (중요 - v2.0 Pro)**:
- 자동 레벨업 (노션 연동)
- 씨앗 농장
- 똑똑한 잔소리
- 통합 대시보드

**P2 (선택 - 향후)**:
- 업적 시스템
- 일일 미션
- 테마 커스터마이징
- 데이터 내보내기/가져오기

## 📝 핵심 기능 요약

1. **직관적인 돌봄**: 밥, 물, 쓰다듬기로 하루치를 돌봄
2. **레벨업 시스템**: 액션에 따른 경험치로 레벨업
3. **리액션**: 기분·상태에 따른 표정·모션 변화
4. **오프닝 & BGM**: 시작 화면과 게임 화면에 각각 배경음
5. **링크 공유**: Vercel 배포 후 링크만 공유해 바로 플레이 가능
