# 하루치 Pro - 라이브 서버 주소

기본 버전과 프로 버전의 라이브 서버 주소입니다.

- **프로 버전 (Pro):** [https://haruchipro.vercel.app](https://haruchipro.vercel.app)
- **기본 버전 (Basic):** [https://haruchi.vercel.app](https://haruchi.vercel.app)

## 로컬 브라우저 접속 주소 (Live Server 등)

로컬에서 테스트할 때는 주소 뒤에 `?tier=pro` 를 붙여야 프로 기능이 활성화됩니다.

- **프로 버전:** http://localhost:5500/tama/?tier=pro
- **기본 버전:** http://localhost:5501/tama/

---
*참고: 포트 번호(`5500`)는 사용 중인 라이브 서버 설정에 따라 다를 수 있습니다.*
