# 하루치 - 햄스터 다마고치 🐹

> 귀여운 햄스터 **하루치**를 키우는 웹 기반 다마고치 게임입니다.

---

## 🌟 프로젝트 소개

**하루치(Haruchi)**는 90년대 다마고치를 모티브로 한 웹 게임입니다.  
반응형 UI와 피드백 중심 플레이로, PC와 모바일에서 간단히 즐길 수 있습니다.

### 주요 특징
- **직관적인 돌봄**: 밥 주기, 물 주기, 쓰다듬기로 하루치를 돌봅니다
- **레벨업 시스템**: 액션에 따른 경험치로 레벨업
- **리액션**: 기분·상태에 따른 표정·모션 변화
- **오프닝 & BGM**: 시작 화면과 게임 화면에 각각 배경음
- **링크 공유**: Vercel 배포 후 링크만 공유해 바로 플레이 가능

---

## 🎮 플레이 방법

1. **시작 화면**에서 화면을 터치하여 오프닝 재생
2. **게임 스타트** 영역을 클릭해 게임 시작
3. 하루치에게 **밥**, **물**, **쓰다듬기**를 해 주세요
4. 집, 그릇, 물통 등을 클릭해 다양한 액션 실행
5. A·B·C 버튼으로 추가 액션 수행
6. 더블클릭으로 **통계** 확인

## 🛠 기술 구성

| 구분 | 내용 |
|------|------|
| **프론트** | HTML, CSS, JavaScript (바닐라) |
| **데이터** | 브라우저 localStorage (클라이언트 저장) |
| **배포** | Vercel (정적 호스팅) |
| **폰트** | Press Start 2P, Noto Sans KR |

- 별도 백엔드·DB 없음
- 노션 연동 미사용 (독립 실행 버전)

## 🚀 실행 및 배포

### 로컬 실행
- `tama/index.html`을 브라우저로 열기 (Live Server 권장)

### Vercel 배포
1. GitHub에 저장소 푸시
2. [Vercel](https://vercel.com)에서 저장소 연결
3. `vercel.json`에 따라 `tama` 폴더가 배포 루트로 설정됨

자세한 절차는 **배포및공유.md**를 참고하세요.
