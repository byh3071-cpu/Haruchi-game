# 🐹 하루치 노션 템플릿 사용 가이드

> 노션 페이지 안에서 하루치를 키우면서 할 일을 관리해 보세요!

---

## 📋 빠른 시작 (3분 설정)

### 1단계: 템플릿 복제
- 이 페이지를 내 노션 워크스페이스에 **복제(Duplicate)** 하세요
- 또는 빈 페이지에서 아래 가이드를 따라 직접 만들 수 있어요

### 2단계: 하루치 임베드 추가
1. 노션에서 `/embed` 를 입력하세요
2. URL에 아래 주소를 붙여넣기:
   ```
   https://haruchi.vercel.app/?notion=1
   ```
3. 임베드 블록의 높이를 **450px 이상**으로 드래그해서 조절하세요
4. 하루치가 바로 나타납니다! 🐹

### 3단계: 오늘 할 일 적기
- 하루치 아래에 체크박스(`/to-do`)로 할 일을 적으세요
- 끝나면 ✅ 체크! 하루치에게 밥도 주고, 쓰담쓰담도 해 주세요

---

## 📐 권장 페이지 구조

```
🐹 하루치와 함께하는 오늘           ← 페이지 제목
├─ 🎯 오늘의 목표 (Callout)        ← 핵심 한 가지
├─ 🐹 하루치 상태 (Embed)          ← haruchi.vercel.app/?notion=1
├─ ✅ 오늘 할 일 (To-do list)      ← 체크하면 하루치가 자라요
├─ 📝 하루 일기 (Toggle heading)   ← 세 줄이면 충분
└─ 🌱 씨앗 농장 (Callout)          ← 연속 출석 기록
```

### 섹션별 안내 문구 (복사해서 쓰세요)

| 섹션 | 캡션 텍스트 |
|------|-----------|
| 🎯 오늘의 목표 | *하나만 정하면 돼요. 오늘 꼭 해내고 싶은 한 가지는?* |
| ✅ 오늘 할 일 | *작은 것부터 하나씩. 체크할 때마다 하루치가 자라요.* |
| 📝 하루 일기 | *세 줄이면 충분해요. 오늘 하루, 어땠나요?* |
| 🌱 씨앗 농장 | *매일 한 가지라도 체크하면, 씨앗이 해바라기가 돼요.* |

## 📱 모바일 사용 팁

- **노션 앱**에서도 임베드가 정상 동작합니다
- 출퇴근길 지하철에서 노션 앱을 열고:
  1. 할 일 체크
  2. 하루치 밥 주기
  3. 일기 세 줄 적기
- 모바일에서는 자동으로 1칼럼 레이아웃으로 전환됩니다

## ❓ 자주 묻는 질문

**Q: 데이터가 날아가지 않나요?**  
A: 브라우저 localStorage에 자동 저장됩니다. 같은 기기/브라우저에서는 유지돼요.

**Q: 여러 기기에서 동기화되나요?**  
A: 기본 버전은 기기별 독립 저장입니다. Pro 버전은 노션 DB를 통해 XP가 동기화됩니다.

**Q: 임베드가 안 보여요!**  
A: `/embed`로 URL을 정확히 입력했는지 확인하세요. `?notion=1` 파라미터가 반드시 포함되어야 합니다.
