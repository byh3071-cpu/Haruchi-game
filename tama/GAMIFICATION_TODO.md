# 게이미피케이션 추가 기능 TODO

## 🎯 우선순위별 구현 가능한 기능들

### ⚡ 빠르게 구현 가능 (30분~1시간)

#### 1. 통계 시스템 (Statistics)
- **구현 내용**: 총 액션 횟수, 총 획득 EXP, 플레이 일수 추적
- **저장**: localStorage에 통계 데이터 추가
- **표시**: 디버그 영역 또는 숨겨진 통계 화면
- **효과**: 사용자가 얼마나 플레이했는지 확인 가능

#### 2. EXP 획득 시 파티클 효과
- **구현 내용**: 버튼 클릭 시 "+10", "+5" 같은 텍스트가 위로 떠오르는 효과
- **CSS 애니메이션**: 간단한 transform + opacity 애니메이션
- **효과**: 액션에 대한 즉각적인 피드백 제공

#### 3. 연속 플레이 보너스 (Combo System)
- **구현 내용**: 짧은 시간 내 연속 액션 시 추가 EXP 보너스
- **예시**: 5초 내 연속 액션 시 +2 EXP 보너스
- **효과**: 빠른 플레이를 장려

#### 4. 레벨업 시 더 화려한 효과
- **구현 내용**: 레벨업 시 화면 전체 플래시, 별 효과 등
- **CSS**: keyframe 애니메이션으로 화면 깜빡임
- **효과**: 레벨업의 성취감 강화

### 🎮 중간 난이도 (1~2시간)

#### 5. 업적/배지 시스템 (Achievements)
- **구현 내용**: 
  - "첫 만남" (처음 시작)
  - "레벨 10 달성"
  - "100번 액션"
  - "하루 종일 플레이"
- **저장**: localStorage에 업적 배열 저장
- **표시**: 업적 달성 시 토스트 알림
- **효과**: 장기적인 목표 제공

#### 6. 일일 미션/퀘스트
- **구현 내용**: 매일 새로운 목표 (예: "오늘 5번 밥 주기")
- **저장**: 날짜별 미션 상태 저장
- **보상**: 미션 완료 시 추가 EXP
- **효과**: 매일 돌아올 이유 제공

#### 7. 통계 화면 (모달)
- **구현 내용**: 특정 키(예: 'S') 누르면 통계 화면 표시
- **표시 내용**: 
  - 총 플레이 시간
  - 총 액션 횟수
  - 레벨별 통계
  - 업적 목록
- **효과**: 사용자가 자신의 진행 상황을 한눈에 확인

#### 8. 쿨타임 시스템
- **구현 내용**: 각 액션마다 쿨타임 추가 (예: 3초)
- **시각적 표시**: 버튼에 타이머 표시
- **효과**: 스팸 클릭 방지, 더 전략적인 플레이

### 🎨 고급 기능 (2~3시간)

#### 9. 사운드 효과
- **구현 내용**: Web Audio API로 간단한 효과음
- **효과음**: 
  - 버튼 클릭
  - EXP 획득
  - 레벨업
  - 배경음악 (선택적)
- **효과**: 몰입감 향상

#### 10. 애니메이션 개선
- **구현 내용**: 
  - 햄스터가 더 다양한 반응 (기쁨, 놀람 등)
  - 배경 효과 (별, 하트 등)
  - 레벨에 따른 햄스터 크기 변화
- **효과**: 시각적 재미 증가

#### 11. 다크모드/테마 시스템
- **구현 내용**: 여러 테마 선택 가능
- **저장**: localStorage에 테마 설정 저장
- **효과**: 개인화된 경험

#### 12. 내보내기/가져오기 기능
- **구현 내용**: 데이터를 JSON으로 내보내기/가져오기
- **효과**: 백업 및 공유 가능
