# 쳇바퀴 달리는 햄스터 애니메이션 가이드

## 🎯 목표
부드럽고 자연스러운 쳇바퀴 달리기 애니메이션 구현

## 📐 이미지 제작 규격

### 1. 스프라이트 시트 방식 (권장)
- **레이아웃**: 가로로 6-8프레임을 나란히 배치
- **각 프레임 크기**: 300x300px (또는 현재 햄스터 크기에 맞춤)
- **전체 이미지 크기**: 
  - 6프레임: 1800x300px
  - 8프레임: 2400x300px
- **배경**: 투명 또는 현재 배경색과 일치

### 2. 프레임 구성 (6프레임 기준)

```
[프레임1] [프레임2] [프레임3] [프레임4] [프레임5] [프레임6]
  ↓        ↓        ↓        ↓        ↓        ↓
왼발내림  양발중간  오른발내림 양발중간  왼발내림  양발중간
```

**각 프레임의 특징:**
- **프레임 1**: 왼쪽 앞발이 쳇바퀴에 닿고, 오른쪽 앞발이 뒤로 뻗어있음
- **프레임 2**: 양 발이 중간 위치, 몸이 약간 위로 올라감
- **프레임 3**: 오른쪽 앞발이 쳇바퀴에 닿고, 왼쪽 앞발이 뒤로 뻗어있음
- **프레임 4**: 양 발이 중간 위치, 몸이 약간 위로 올라감
- **프레임 5**: 프레임 1과 유사하지만 미묘한 차이 (속도감 표현)
- **프레임 6**: 프레임 2와 유사하지만 미묘한 차이

### 3. 시점 통일
- **모든 프레임을 옆모습으로 통일** (현재 normal.png와 동일한 시점)
- 햄스터가 오른쪽을 향해 달리는 모습
- 쳇바퀴는 햄스터와 함께 그려지되, 회전 효과는 CSS로 처리

## 🎨 제미나이 프롬프트 (6프레임 버전)

### 프롬프트 1: 기본 버전
```
픽셀 아트 스타일로 쳇바퀴를 타는 햄스터 애니메이션 스프라이트 시트를 만들어줘.

요구사항:
- 가로로 6개 프레임을 나란히 배치
- 각 프레임 크기: 300x300px
- 전체 이미지 크기: 1800x300px
- 배경은 투명
- 픽셀 아트 스타일, 현재 normal.png와 동일한 스타일 유지

프레임 구성:
1. 왼쪽 앞발이 쳇바퀴에 닿고, 오른쪽 앞발이 뒤로 뻗어있음
2. 양 발이 중간 위치, 몸이 약간 위로 올라감
3. 오른쪽 앞발이 쳇바퀴에 닿고, 왼쪽 앞발이 뒤로 뻗어있음
4. 양 발이 중간 위치, 몸이 약간 위로 올라감
5. 프레임 1과 유사하지만 미묘한 차이
6. 프레임 2와 유사하지만 미묘한 차이

시점: 모든 프레임을 옆모습으로 통일 (햄스터가 오른쪽을 향함)
쳇바퀴: 각 프레임에 나무 질감의 쳇바퀴 포함, 햄스터가 타고 있음
색상: 현재 normal.png와 동일한 파스텔 톤 유지
```

### 프롬프트 2: 상세 버전 (더 자연스러운 움직임)
```
픽셀 아트 스타일로 쳇바퀴를 빠르게 달리는 햄스터 애니메이션 스프라이트 시트를 만들어줘.

레이아웃:
- 가로로 8개 프레임을 나란히 배치
- 각 프레임: 300x300px
- 전체: 2400x300px
- 배경: 투명

애니메이션 사이클 (8프레임):
프레임 1: 왼쪽 앞발이 쳇바퀴 바닥에 닿는 순간, 오른쪽 앞발은 공중에, 몸이 약간 낮음
프레임 2: 왼쪽 앞발이 쳇바퀴를 밀고, 오른쪽 앞발이 앞으로 이동, 몸이 약간 올라감
프레임 3: 양 발이 거의 같은 높이, 몸이 가장 높은 위치
프레임 4: 오른쪽 앞발이 앞으로 뻗어나감, 왼쪽 앞발이 뒤로 이동 시작
프레임 5: 오른쪽 앞발이 쳇바퀴 바닥에 닿는 순간, 왼쪽 앞발은 공중에, 몸이 약간 낮음
프레임 6: 오른쪽 앞발이 쳇바퀴를 밀고, 왼쪽 앞발이 앞으로 이동, 몸이 약간 올라감
프레임 7: 양 발이 거의 같은 높이, 몸이 가장 높은 위치
프레임 8: 왼쪽 앞발이 앞으로 뻗어나감, 오른쪽 앞발이 뒤로 이동 시작

스타일:
- 픽셀 아트, 굵은 검은색 테두리
- 파스텔 톤 색상 (연한 분홍, 베이지, 크림색)
- 모든 프레임 옆모습 (햄스터가 오른쪽을 향함)
- 쳇바퀴는 각 프레임에 포함 (나무 질감, 갈색)
- 햄스터의 눈은 반짝이는 하이라이트 포함
- 볼에 분홍색 홍조
- 입은 작은 곡선 또는 "v" 모양

참고: 현재 normal.png의 스타일과 일치해야 함
```

### 프롬프트 3: 쳇바퀴 분리 버전 (고급)
```
픽셀 아트 스타일로 쳇바퀴를 타는 햄스터 애니메이션을 만들어줘.

파일 1: wheel_hamster.png (햄스터만, 6프레임)
- 가로로 6개 프레임, 각 300x300px
- 배경 투명
- 햄스터만 그리기 (쳇바퀴 제외)
- 옆모습, 오른쪽을 향함
- 달리기 사이클: 왼발내림 → 중간 → 오른발내림 → 중간 → 반복

파일 2: wheel_base.png (쳇바퀴만, 1프레임)
- 300x300px
- 배경 투명
- 나무 질감의 쳇바퀴만 그리기
- 햄스터 없음
- 회전 효과는 CSS로 처리

스타일: normal.png와 동일한 픽셀 아트 스타일
```

## 🔧 코드 수정 필요사항

### 6프레임 사용 시
```css
.ham.wheel{
  background-size: 600% 100%; /* 6프레임 */
  animation: wheelRun 0.8s steps(6) infinite;
}
```

### 8프레임 사용 시
```css
.ham.wheel{
  background-size: 800% 100%; /* 8프레임 */
  animation: wheelRun 0.8s steps(8) infinite;
}
```

## 💡 추가 팁

1. **프레임 수가 많을수록 부드러움**: 6프레임은 최소, 8프레임 이상 권장
2. **속도 조절**: CSS의 `animation-duration`으로 조절 가능 (0.6s~1.2s)
3. **테스트**: 이미지 생성 후 실제로 애니메이션을 테스트해보고 미세 조정
4. **참고 이미지**: normal.png를 참고하여 스타일 일치시키기
